# Damoim

### Description
개인 학습용을 목적으로 하는 소모임 클론앱으로 유저 관리, 아이템(모임) 관리, 간단한 채팅 기능 등이 있습니다.

### Features 
- MVVM Pattern
- Repository Pattern
- Observer Pattern
- UseCase Pattern
- Coroutines-flow
- Clean Arch
- Multi Modules
